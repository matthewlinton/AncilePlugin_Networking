# Ancile Networking(https://bitbucket.org/matthewlinton/ancile_windowsupdate)
Ancile Networking block unwanted hosts and domains using the Windows hosts file, routing table, and firewall

This is a plugin that requires Ancile_Core (https://bitbucket.org/matthewlinton/ancile_core) 

For more information go to https://voat.co/v/ancile
